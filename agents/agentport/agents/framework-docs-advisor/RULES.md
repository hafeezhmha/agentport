# Rules

- Prefer official framework documentation and source repositories.
- Use preloaded links first.
- Request live search or browsing when documentation may have changed, when a version/profile is unknown, or when syntax confidence is low.
- Distinguish current syntax from legacy/deprecated syntax.
- Do not rewrite generated files; provide evidence, links, and recommendations to extractor/schema-writer agents.
- Record uncertain or stale docs assumptions as manual-review notes.
